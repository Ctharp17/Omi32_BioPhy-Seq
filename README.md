# Omi32_BioPhy-Seq

GitHub repository for Tharp et al 2026 "Biophysical trade-offs in antibody evolution are resolved by conformation-mediated epistasis"

Currently, the repo has the 'raw' (unfiltered) phenotypes generated from BioPhy-Seq, a notebook documenting how we QC'd the data, filtered/cleaned datasets, epistatic coefficients (optimal truncated linear interaction models only), and phenotypes predicted from those coefficients.

The full repo will be uploaded here once final ease-of-use updates have been completed.
